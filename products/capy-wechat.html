MatteoCui001/capy-wechat TypeScript Bun

What it is

一个 TypeScript bot，通过腾讯官方的 ClawBot iLink API 把你的微信账号接入 Claude AI。不是逆向，不是爬虫，走的是正规 API 渠道。

支持文字、语音、图片、文件、视频全媒体类型，内置 typing indicator，还有一个「干活模式」——切换后直接运行 Claude Code，让 AI 真正帮你执行任务。

Installation

bash 
# 需要 Bun runtime
curl -fsSL https://bun.sh/install | bash

# 克隆项目
git clone https://github.com/MatteoCui001/capy-wechat.git
cd capy-wechat

# 安装依赖
bun install

# 首次认证（扫二维码登录）
bun setup.ts

# 启动
AI_GATEWAY_API_KEY=<your-key> bun src/main.ts

Mode Switching

在微信聊天中直接发送关键词切换模式

关键词 切换到 说明

休闲 聊天 聊天模式
